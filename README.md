# 🎓 Future Platform – Student Information System (SIS)

## 📌 Overview
This project is a **Student Information System (SIS)** designed for the Future Platform.

The system aims to digitize academic operations including:
- Student data management  
- Academic progress tracking  
- GPA calculation  
- Instructor grading  

> 🚀 This version represents the **Minimum Viable Product (MVP)** focusing on core academic workflows.

---

## 🎯 MVP Scope

### ✅ Included Features
- 🔐 Authentication System (Student & Admin)
- 👨‍🎓 Student Dashboard (Academic data & GPA)
- 👨‍🏫 Instructor Dashboard (Courses & grading)
- 🏫 Admin Dashboard (Student management & operations)
- 📊 GPA Calculation Engine

### ❌ Out of Scope (Future Phases)
- Course Registration (Enrollment System)
- E-Learning (Videos, Assignments, Quizzes)
- Payment / Financial Systems
- Mobile Application
